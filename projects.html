Placement Data Analysis

Developed a machine learning based Placement Data Analysis system using Python. Collected and cleaned student academic datasets and performed exploratory data analysis using Pandas, NumPy and Matplotlib to identify important placement factors. Built predictive models using Scikit-learn to estimate placement probability based on academic performance and skills. The project demonstrates practical applications of machine learning and predictive analytics for career outcome prediction.

The system helps in early detection and supports healthcare professionals in making informed decisions.
